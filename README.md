## Overview
`jscalculator` is a simple web-based calculator built using JavaScript. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator features a user-friendly interface with buttons for numbers and operations.

## Features
- **Basic Operations**: Supports addition, subtraction, multiplication, and division.
- **Clear Functionality**: Users can clear the current input and previous calculations.
- **Delete Functionality**: Users can delete the last entered digit.
- **Decimal Support**: Users can input decimal numbers.

## Usage
1. Include the `calculator.js` script in your HTML file.
2. Ensure that the HTML structure includes elements with the following classes:
   - `.curr-display`: For displaying the current input.
   - `.prev-display`: For displaying the previous calculations.
   - `.number`: For number buttons (0-9).
   - `.operation`: For operation buttons (+, -, *, /).
   - `.clear`: For the clear button.
   - `.delete`: For the delete button.
   - `.equal`: For the equal button.
